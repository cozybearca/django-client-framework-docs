ObjectManager<T: `Model`_>¶

A generic class where T is constrained by Model. This class provides the ability to modify an model object. When constructing an ObjectManager instance, an model object is passed into the ObjectManager class constructor. Any field that’s available on the model objects becomes available on the ObjectManager object. In addition, the ObjectManager object provides methods that are used to modify the wrapped model object, and save the changes to the backend.

Consturctor¶

Pass the model object into the ObjectManager constructor.

Warning

Most of time, you should use the ObjectManager returned by `CollectionManager.page(...)`_ or `CollectionManager.get(...)`_, instead of creating one yourself.

Example¶

let product_model = new Product()
let product = new ObjectManagerImpl(product_model)

Methods¶

.refresh()¶

Pulls data from the server and updates the local data. It sends a GET request to the server.

let product: ObjectManager<Product> = ...
product.barcode = 'xxyy'
// GET /product/1
await product.refresh() // resets product barcode

.save()¶

Updates the server with properties that have been changed on the object. It sends a PATCH request to the server.

let product: ObjectManager<Product> = ...
product.barcode = "xxyy"
product.price = null
product.brand_id = 2
// PATCH /product/1 {"barcode": "xxyy", "price": null, "brand_id": 2}
await product.save()

.update()¶

Updates the server with a dictionary, and also saves the changes locally. If the dictionary contains a key that is not a property name, the method throws an error. It sends a PATCH request to the server.

let product: ObjectManager<Product> = ...
// PATCH /product/1 {"barcode": "xxyy", "price": null}
await product.update({
    barcode: "xxyy",
    price: null
)}

.delete()¶

Removes the data from the server. It sends a DELETE request to the server.

let product: ObjectManager<Product> = ...
// DELETE /product/1
await product.delete()
